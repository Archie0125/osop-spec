# OSOP Adopters

Projects and tools that support OSOP.

| Project | Conformance | Version | Capabilities |
|---|---|---|---|
| [osop](https://github.com/user/osop) | Execute | 1.0 | parse, validate, execute, report |
| [osop-editor](https://github.com/user/osop-editor) | Validate | 1.0 | parse, validate, render, report |
| [osop-sdk-js](https://github.com/user/osop-sdk-js) | Parse | 1.0 | parse, validate, report |
| [osop-sdk-py](https://github.com/user/osop-sdk-py) | Parse | 1.0 | parse, validate, report |

## How to add your project
Submit a PR adding your entry to the table above. Include:
- A `.osop-compat.json` in your repo root
- Link to your repository
