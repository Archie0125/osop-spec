# OSOP Manifesto

> Let humans understand processes. Let AI run processes.

## Core Beliefs

- Turn AI output from bullet points into contracts, from suggestions into executable systems.
- Every step is traceable, every iteration verifiable, every failure locatable, every optimization comparable.
- Process is the common language across agents: different agents can collaborate as long as they follow the same contract.

## Six Value Propositions

1. **Correct AI logic** — schema + contract forces AI to specify inputs/outputs and success criteria.
2. **Rapid strategy execution** — decompose strategy into nodes, responsibilities, conditions, handoffs.
3. **Human-first visualization** — Graph/Story/Role views let non-engineers understand processes.
4. **Full traceability (Ledger)** — every node has a run record for precise fault localization.
5. **Testable (TDD)** — workflows have tests (unit/integration/e2e/simulation).
6. **Evolvable (AI+Human Governance)** — AI proposes workflow patch/PR, humans review and merge.
